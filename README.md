# Análise de Vendas para E-commerce 

Este é um projeto prático de análise de dados focado em extrair insights de negócios a partir de um conjunto de dados de vendas de e-commerce. O script gera um banco de dados fictício do zero e realiza análises exploratórias para responder a perguntas estratégicas de negócio.

Projeto desenvolvido como parte das práticas do curso *Fundamentos da Linguagem Python para Análise de Dados e IA* (Data Science Academy).

## Objetivos da Análise
O código simula um ambiente real de e-commerce, gerando 500 registros de vendas com datas, produtos, preços e localizações. A partir desses dados brutos, o script realiza limpeza, engenharia de atributos (como o cálculo de Faturamento e Status de Entrega) e responde às seguintes métricas:

1. **Top 10 Produtos Mais Vendidos:** Identificação dos carros-chefes de volume da loja.
2. **Evolução do Faturamento Mensal:** Análise de tendência de receita ao longo do tempo.
3. **Faturamento por Estado:** Mapeamento geográfico de onde vem a maior parte da receita.
4. **Faturamento por Categoria:** Desdobramento financeiro por segmento de produtos (Eletrônicos, Hardware, Móveis, etc.).

## Tecnologias Utilizadas
* **Python 3**
* **Pandas:** Para estruturação, limpeza e manipulação do DataFrame.
* **NumPy:** Para geração de dados aleatórios e operações matemáticas.
* **Matplotlib & Seaborn:** Para a construção de visualizações de dados claras e elegantes.
